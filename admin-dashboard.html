Admin Dashboard

Available Admin API Endpoints

POST /api/admin/login - Admin login
GET /api/admin/dashboard/stats - Get dashboard statistics
GET /api/admin/users - Get all users with pagination
GET /api/admin/users/:userId - Get user details
PUT /api/admin/users/:userId/status - Update user status
GET /api/admin/payments - Get all payments
POST /api/admin/payments/:paymentId/refund - Process refund
GET /api/admin/memberships - Get all memberships
GET /api/admin/messages - Get contact messages
GET /api/admin/analytics - Get analytics data
GET /api/admin/quick-stats - Get quick stats for dashboard
GET /api/admin/alerts - Get system alerts
